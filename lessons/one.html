WordPress is open source software you can use to create a beautiful website, blog, or app.

Install DesktopServer

DesktopServer was created specifically to help WordPress designers or developers spin up local building and testing environments. We will be using DesktopServer to create a local server that will host our local WordPress site. Please visit: https://serverpress.com/, and download the free version of DesktopServer.

Follow the following instructions to install DesktopServer: http://docs.serverpress.com/article/150-getting-started-with-desktopserver.

Install Atom or Sublime

Atom is a text editor that's modern, approachable, yet hackable to the core—a tool you can customize to do anything but also use productively without ever touching a config file.

Sublime Text is a sophisticated text editor for code, markup and prose.You'll love the slick user interface, extraordinary features and amazing performance.

  • Download Atom: https://atom.io/
  • Download SublimeText3: https://www.sublimetext.com/3
